# Reference - AI Agent Guidelines

## Purpose

Provide consistent terminology across the project.

## Glossary (glossary.md)

The main reference containing:

- Technical terms
- Mathematical concepts
- Game mechanics terminology
- Abbreviations

## Guidelines

### Adding Terms

1. Alphabetize entries
2. Provide clear definition
3. Include context/usage
4. Cross-reference related terms

### Updating Definitions

- Keep definitions concise
- Ensure accuracy
- Update when usage changes
- Note deprecated terms

### Consistency

The glossary defines canonical terminology:

- Use these terms in documentation
- Link to glossary for definitions
- Maintain consistent usage
