## Quick Reference

### Core Terms

| Term | Definition |
|------|------------|
| **Quadray** | Four-dimensional coordinate system based on tetrahedral geometry |
| **Tetrahedral Voxel** | A tetrahedral volume element (vs cubic voxel) |
| **CCP** | Close-Centered Packing - efficient ball arrangement |
| **Chunk** | Section of world for efficient management |

### Shape Types

| Shape | Description |
|-------|-------------|
| **Octahedron** | 8-faced polyhedron that fills space with tetrahedra |
| **Tetrahedron Z** | One orientation of 4-faced tetrahedron |
| **Tetrahedron C** | Other orientation of 4-faced tetrahedron |

### Coordinate Systems

| System | Components |
|--------|------------|
| **Cartesian** | (x, y, z) - traditional 3D |
| **Quadray** | (a, b, c, d) - tetrahedral 4D |
