# Reference Papers - AI Agent Guidelines

## Purpose

Academic and technical reference documents providing deeper theoretical foundation.

## Document Types

- Technical papers
- Formal analysis
- Research documentation

## Guidelines

### Using These Documents

- Reference for theoretical understanding
- Citation source for documentation
- Implementation guidance

### Updating

These are largely reference documents:

- Update sparingly
- Maintain academic style
- Add new papers for new research
- Don't modify original analysis conclusions

### Adding New Papers

1. Use descriptive filename
2. Include abstract/overview
3. Add to README.md
4. Note date and context

## Note

Filenames are truncated due to long original titles. Content is complete within files.
