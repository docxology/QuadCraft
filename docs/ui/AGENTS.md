# UI Documentation - AI Agent Guidelines

## Purpose

Documents the user interface design and implementation.

## Documents

### user_interface.md

Overview of UI components:

- Panel layouts
- Control organization
- Information displays

### interface_design.md

Detailed design specifications:

- Styling guidelines
- Component specifications
- Interaction patterns

## Guidelines

### Design Changes

1. Consider accessibility
2. Maintain dark theme consistency
3. Test responsiveness
4. Document in these files

### Adding UI Components

1. Follow existing patterns
2. Use consistent styling
3. Document in interface_design.md
4. Consider keyboard shortcuts

### CSS Updates

When modifying styles:

- Update documentation
- Test all panels
- Verify contrast/readability
- Consider mobile (if applicable)
