# Styles

CSS styling for the modular browser version.

## Files

| File | Purpose |
|------|---------|
| `main.css` | Complete styling for the application |

## main.css

Comprehensive styling including:

### Layout

- Three-panel layout (controls, analysis, game)
- Responsive sizing
- Panel headers and sections

### Components

- Button styles
- Input fields and sliders
- Canvas containers
- Collapsible sections

### Theme

- Dark mode color scheme
- Consistent color palette
- Proper contrast for readability

### Responsive

- Flexbox-based layout
- Panel resizing support
- Mobile considerations

## Usage

```html
<link rel="stylesheet" href="styles/main.css">
```

## CSS Variables

```css
:root {
  --bg-primary: #1a1a2e;
  --bg-secondary: #16213e;
  --text-primary: #eee;
  --accent-color: #4a90d9;
  /* ... etc */
}
```
