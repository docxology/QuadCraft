## Var.js

A comprehensive JavaScript utility library providing extended functionality.

### Features

- Enhanced variable types
- Mathematical operations
- Data structure utilities
- Computation helpers

### Usage

```html
<script src="lib/Var.js"></script>
```

Then access via global namespace or module imports.

## Notes

- Large file; consider lazy loading if not immediately needed
- Used by main game application
- Provides foundational utilities for game logic
