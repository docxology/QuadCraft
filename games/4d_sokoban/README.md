# 4D Sokoban

## Overview
Box-pushing puzzle on 4D IVM grid with 12 kissing-sphere directions. Part of [QuadCraft](../).

## Quick Start
To play this game, run the following from the project root:

```bash
python3 games/run_games.py --game sokoban
```

- **Game Index**: [Index](../GAMES_INDEX.md) - List of all QuadCraft games.

Run the test suite for this game:
